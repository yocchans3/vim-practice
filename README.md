# vim-practice

## 移動
:set number

|コマンド|内容|
|---|---|
|G|ファイルの末尾に移動|
|gg|ファイルの先頭に移動|
|$|行の最後に移動|
|0|行の先頭へ移動(インデント無視して先頭へ)|
|^|行の先頭へ移動|
|{|段落毎に上へ移動|
|}|段落毎に下へ移動|
|ctrl+o|移動前に戻る|
|w|単語の先頭に進む|
|e|単語の先頭に戻る|

## 削除
|コマンド|内容|
|---|---|
|dd|行を削除|
|dw|単語を削除(カーソル先頭)|
|x|1文字削除|
|u|undo|
|ctrl+r|redo|
|i|インサートモード|
|a|次の文字からインサートモード|
